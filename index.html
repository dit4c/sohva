Gnieh Sohva

Gnieh Sohva is a set of scala libraries allowing users to easily store and retrieve data from couchdb databases.

It consists in several tools:

  • sohva-client is a client allowing user to query a CouchDB server,
  • sohva-entities is a small framework allowing to manage entity/component systems stored in a CouchDB database,
  • sohva-testing is a module that allows to test applications that use CouchDB.

Have a look at the wiki pages for more documentation
